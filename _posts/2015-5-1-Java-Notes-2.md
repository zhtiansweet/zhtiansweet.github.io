### Data Type
* Primitive Type: stored in the stack (_"Here"_)
  * Numeric Type
    * Integer: **byte (1b), short (2b), int (4b), long (8b)**
    * Float: **float, double**
  * **char**
  * **boolean**
* Reference Type: pointer to the object stored in the heap (_"There"_)
  * **class**
  * **interface**
  * **Array**
